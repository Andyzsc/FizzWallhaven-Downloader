# FizzWallhaven 2.1

FizzWallhaven is a Windows wallpaper downloader for Wallhaven.

## Features

- Search wallpapers from Wallhaven
- Preview wallpapers by page
- Download current preview page
- Batch download 100/500+ wallpapers
- Support sorting: Latest, Hot, Toplist, Random
- Support categories: General, Anime, People
- Support purity filters
- Support resolution and ratio filters
- Two-stage download strategy for rate-limit retry

## Requirements

- Python 3.12+
- requests
- customtkinter
- pillow

## Run from source

```bash
pip install -r requirements.txt
python FizzWallhaven2.1.py
